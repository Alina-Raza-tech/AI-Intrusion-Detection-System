# AI-Powered Intrusion Detection System

## Overview
This project focuses on developing an AI-based Intrusion Detection System (IDS) using the CICIDS 2017 dataset to detect cyber threats in real-time network environments.

## Dataset
- CICIDS 2017 (Real-world dataset)
- 2.8M+ records
- 80+ features

## Models Used
- Random Forest
- XGBoost

## Key Results
- Achieved 99.95% accuracy
- PCA reduced features by 67%
- SMOTE improved minority attack detection

## Techniques
- Machine Learning
- Data Preprocessing
- PCA
- SMOTE
- Feature Engineering
